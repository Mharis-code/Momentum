# Content Creation System

**Description:** Build a repeatable, AI-assisted system for producing consistent content across multiple platforms.
**Status:** Active — Planning and building phase
**Key dates:** No hard deadline. Must be running before any heavy audience growth push.

## Channels (Priority Order)

1. LinkedIn — professional authority, primary channel
2. TikTok and Instagram — reach and awareness
3. Skool communities — trust and engagement
4. YouTube — long-form, once short-form is consistent

## System Vision

- Research niche topics relevant to solopreneurs and AI
- Write LinkedIn posts with strong hooks
- Repurpose content across platforms
- Generate visuals via Canva
- Automate as much as possible with AI agents
